# Toyota Used Car Price Analysis

## Overview

This project performs Exploratory Data Analysis (EDA) on a Toyota Used Cars dataset to understand the factors affecting the resale price of used Toyota cars.

The analysis includes:
- Data Cleaning
- Missing Value Handling
- Data Visualization
- Correlation Analysis
- Business Insights


## Dataset

The dataset contains information about **1,436 Toyota used cars** with features such as:

- Price
- Age
- Kilometers Driven
- Fuel Type
- Horsepower
- Color
- Transmission
- Engine Capacity
- Doors
- Weight


## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab


## Exploratory Data Analysis

The following analyses were performed:

- Price Distribution
- Fuel Type Analysis
- Age Distribution
- Price vs Age
- Price vs Kilometers Driven
- Price vs Horsepower
- Correlation Heatmap


## Key Insights

- Most used cars are priced between **₹8,000–₹12,000**.
- Petrol cars dominate the dataset.
- Older cars generally have lower resale values.
- Higher mileage is associated with lower selling prices.
- Age has the strongest negative correlation with price.
